# Community Educational Platform

Educational platform deployed using modern DevOps infrastructure.

## Technologies Used

- AWS EC2
- Terraform
- Ansible
- Docker
- Linux Ubuntu
- SSH
- Docker Compose

## Features

- Automated infrastructure deployment
- Configuration management
- Containerized services
- Cloud deployment
- Community network oriented architecture

## Architecture

User → AWS EC2 → Docker → Educational Web Platform

## Deployment

Terraform creates:
- EC2 instance
- Security Groups

Ansible configures:
- Docker
- Docker Compose
- Web service deployment

## Local Network Vision

The platform can later be replicated inside community mesh networks using OpenWrt routers.
